# Tipos Primitivos em PHP

* escalares

* compostos

* especiais

## Tipos Primitivos Escalares

**String**: sequencia de letras, números e símbolos, sempre representadas entre aspas.
```php
$sobrenome = "Nome"
```

**Int ou Integer**: Um valor numérico Inteiro, aquele que vem sem a oarte decimal.
```php
$idade = 00
```

**Float ou Double**: Um valor numérico Real, que vem com a parte decimal, depois do ponto flutuante.
```php
$peso = 00.00
```

**Bool ou Boolean**: Um valor lógico ou Booleano, que aceita apenas os valores verdadeiro ou falso (**true** e **false**).
```php
$casado = false
```
